# valor-capex
A standalone Capex billing and amortization module built with Next.js, Node.js, and PostgreSQL — calculates monthly asset billing (depreciation, interest, maintenance, contingency, and management fees) and supports Excel import/export.

Valor Capex is a web application for managing capital expenditure (Capex) billing. Users can add capital assets or bulk-import them from an Excel sheet, and the system automatically calculates the monthly bill for each asset — including amortization, interest on capex, maintenance, contingency cost, and management fees. Data is stored in PostgreSQL and displayed in an interactive dashboard. Built with Next.js (App Router), Node.js, Prisma, and PostgreSQL.
